В папке лежит датасет по проблемам с сердцем. Обработайте его, выделите важные столбцы и обучите модель классификации, которая будет определять есть-ли у человека проблемы с сердцем.

Sex - Пол;	1 = мужчина, 0 = женщина
chest pain type	-Тип боли в груди;  0 = типичная стенокардия, 1 = атипичная стенокардия, 2 = неангинозная боль, 3 = бессимптомная
resting blood pressure	- Артериальное давление в покое; Измеряется в мм рт. ст. при поступлении в стационар
serum cholesterol	 - Уровень холестерина в сыворотке; Измеряется в мг/дл
fasting blood sugar > 120 mg/dl	 - Уровень сахара в крови натощак > 120 мг/дл; 1 = истинно (уровень выше), 0 = ложно (уровень ниже или равен)
resting electrocardiographic results - Результаты ЭКГ в покое; 0 = норма, 1 = отклонения сегмента ST-T, 2 = признаки гипертрофии левого желудочка
max heart rate achieved - Максимальная ЧСС при нагрузке; Достигнутая максимальная частота сердечных сокращений (уд/мин)
exercise induced angina - Стенокардия, вызванная нагрузкой; 1 = да, 0 = нет
oldpeak - Депрессия сегмента ST при нагрузке; Величина депрессии сегмента ST, вызванной нагрузкой, относительно покоя
slope - Наклон сегмента ST при пиковой нагрузке; 0 = восходящий, 1 = горизонтальный, 2 = нисходящий
ca	- Количество крупных сосудов, окрашенных при флюороскопии; Число от 0 до 3
thal	- Результат талассемического теста
target - Наличие сердечного заболевания; 0 = заболевание отсутствует, 1 = заболевание присутствует

В данном датасете ключевой параметр - target
Затем выберите 3 модели классификации и обучите их на данном датасете. Сравните их точность.
